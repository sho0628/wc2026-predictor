# WINNER Predictor — FIFA World Cup 2026 試合予測アプリ

楽天WINNER（スポーツくじ）でのベットを支援する、W杯2026の試合予測Webアプリです。
**当たりやすさ**ではなく**期待値（EV）**で「買い目」を見つけることに主眼を置いています。

## 起動方法

1. このフォルダ（`WINNER-app`）の中の **`index.html`** をダブルクリック
2. お使いのブラウザ（Edge / Chrome 等）で開きます。インストール不要。

> 国旗画像のみインターネット接続が必要です（オフラインでも旗が出ないだけで全機能は動作します）。

## できること

| タブ | 内容 |
|---|---|
| **試合予測** | 2チームを選ぶ → 勝/分/負の確率・予想スコア・Over/Under・両者得点。さらに **WINNER 18択それぞれの的中確率とフェア倍率**、払戻倍率を入れると**期待値で買い目を判定**。 |
| **日程・組** | 全12組・グループ72試合。試合をクリックすると予測タブに読み込み（高地・開催国補正を自動適用）。 |
| **順位予測** | 各組を1万回モンテカルロ → 1位・突破（上位2）・3位確率。 |
| **優勝予想** | 大会全体をシミュレート → 各国の優勝・決勝・ベスト4到達確率（WINNER優勝予想の参考）。 |
| **レーティング** | Elo・FIFAランク・直近フォーム・主力/負傷の一覧。 |
| **使い方・方法** | モデルの仕組みとWINNERの正しい仕様。 |

## WINNERの仕様について（重要）

WINNERの「1試合予想」は**勝/分/負ではなくスコア予想（18択）**で、**変動オッズ（パリミュチュエル）**です。
払戻倍率は締切まで動き、**人気のない選択肢ほど倍率が高く**なります。モデルの的中確率が
払戻の含意確率を上回る（=払戻倍率 > フェア倍率）選択肢が、理論上の「買い目」です。

## 予測モデル

- **World Football Elo** を主軸（+ 直近フォーム + 開催国ホーム/高地補正 → 実効Elo）
- Elo差 → 勝点期待値 `We = 1/(1+10^(-差/400))`
- **Dixon-Coles 補正付きポアソン**でスコア確率行列を構築 → 18択・1X2・スコアを一貫導出
- 期待値 `EV = 確率 × 倍率 − 1`、資金管理は**1/4ケリー**を推奨

## データ更新

最新化したいときは `data.js` の `teams`（elo / form / inj）を編集してください。
直近の親善試合結果や負傷情報を反映すると精度が上がります。
